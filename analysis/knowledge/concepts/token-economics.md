# Token Economics

## Definition

Token economics covers how firms budget, meter, route, gate, and justify AI consumption relative to workflow value.

## Evidence Summary

- The seed evidence supports a narrow claim that some large organizations segment budgeting by use-case type and add rollout gates to limit burn.
- The current support is adequate for a draft synthesized claim, not for a settled benchmark item.
- A second draft claim captures ring-fenced experimentation budget as a mono-source governance pattern, not a recurring corpus-wide rule.

## Open Questions

- Which observed patterns are cost governance versus workflow quality governance?
- How often do firms manage at the model, workflow, seat, or business-unit level?
- When should token economics be scored under Operating Capabilities versus Infrastructure telemetry?
