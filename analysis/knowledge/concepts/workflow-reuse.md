# Workflow Reuse

## Definition

Workflow reuse describes how AI-enabled workflows, automations, prompts, plugins, or templates move from one local use case to broader organizational use.

## Evidence Summary

- The current evidence supports job-owner-led automation and non-technical role-specific tool building.
- The earlier marketplace framing was narrowed. The current reuse evidence supports local sharing through a plugin repository in one source, not a proven broad marketplace or network-effects claim.

## Open Questions

- Does reuse happen through formal marketplaces, local repositories, templates, peer sharing, or platform-managed catalogs?
- What review is required before a local automation becomes reusable organizational infrastructure?
- Which reuse patterns are startup-skewed versus enterprise-specific implementation forms?
