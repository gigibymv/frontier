# Token Economics

## Short Definition

Token economics covers how firms budget, meter, route, gate, and justify AI consumption relative to workflow value.

## Current Synthesis

The current evidence supports two draft patterns: use-case-segmented token controls and ring-fenced experimentation budget. The first has broader support; the second is mono-source.

## Evidence-Backed Claims

- `clm-oc-token-budgeting-by-use-case`
- `clm-oc-experimentation-budget-governance`

## Important Caveats

- Token budgeting and experimentation budget are related but not identical.
- Current evidence does not yet support a full AI unit-economics benchmark item.
